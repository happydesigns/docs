# `nuxt-variants`

`happydesigns/nuxt-variants` publishes `@happydesigns/nuxt-variants`.

It is a small Nuxt module for building one shared layout that can behave differently per page.

## What it provides

- Flat variant registry.
- Deep merge.
- Inheritance.
- Runtime `app.config.ts` overrides.
- Generated TypeScript types.
- DevTools tab.
- Nuxt Content schema helper.

## Why it exists

It supports reusable layouts, page variants, and content schema composition without turning every page difference into a separate layout or component fork.

The package description is:

> A centralized, deeply-merging configuration engine for Nuxt layouts with build-time variant graph and Nuxt Content v3 schema merging.
