`id` is the product key for Identity.

Identity defines how brand layers are created, structured, documented, and applied on top of reusable Nuxt UI foundations.

Use the product links for installation, examples, and implementation details.

## Product scope

`id` provides reusable brand-layer infrastructure:

- Reusable brand-layer model.
- Token schema direction.
- Scaffolding and templates.
- Identity implementation guides.
- Reusable examples for happydesigns and clients.

## Relationship to `brand`

`brand` is the happydesigns-specific implementation. `id` defines the reusable identity system behind it.
