`happydesigns/business` is the private Business OS product.

Business OS is an automation-first operating system for individuals, small businesses, clubs, and agent-powered companies.

## Product direction

Business OS combines:

- Human UI.
- Reusable domain modules.
- Work queues.
- Knowledge bases.
- Workflows.
- APIs.
- Webhooks.
- MCP access.

The same system should be usable by humans, scripts, external automation platforms, and AI agents.

## Architecture direction

- Nuxt full-stack app.
- Nuxt UI.
- Nuxt layers for branding and customer-specific app variants.
- Nuxt modules for installable behavior and extension points.
- Nitro server routes for APIs, jobs, deployment bindings, and runtime adapters.
- Plain TypeScript packages for deterministic core logic.
- Cloudflare-first managed hosting.
- db0 and Drizzle database foundation.
- D1 first, PostgreSQL later.
- Local and cloud runtime modes as first-class targets.

## MVP areas

The MVP includes organizations, users, memberships, permissions, knowledge-base registry, tasks, work queues, workflow history, agent roles, governance policies, documents, audit log, API foundations, webhook foundations, MCP foundations, companies, contacts, projects, and documents.

Later areas include offers, invoices, E-Rechnung, payments, expenses, and bookkeeping exports.
