`docs` is this Docus site.

It is the central source of truth for happydesigns ecosystem strategy and the default AI entrypoint for architecture, development principles, identity strategy, and product boundaries.

Use this site when a decision affects more than one product or when agents need published operating rules.

Docs are stable, reader-facing, focused, and retrievable by humans and agents.

## What it provides

- Human-readable documentation.
- AI-readable content.
- Docus MCP at `/mcp`.
- LLM files such as `/llms.txt`.
- Installable agent skills from root `skills/`.
