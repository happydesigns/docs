# `help`

`happydesigns/help` is the future customer-facing help center.

It should be non-technical and focused on people using their websites.

## Future topics

- How to edit website content.
- How to use Nuxt Studio.
- How to publish changes.
- How to manage images.
- SEO basics.
- Customer-facing website workflows.

## Boundary

`help` is not where architecture docs, product internals, reusable package details, or agent workflow rules belong.
