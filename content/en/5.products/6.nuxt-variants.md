`happydesigns/nuxt-variants` publishes `@happydesigns/nuxt-variants`.

It is a small Nuxt module for building one shared layout that can behave differently per page.

Use `nuxt-variants` when shared layouts need typed feature composition instead of duplicated layouts and route-specific logic.

Centralize variant decisions in a registry when page-level layout behavior would otherwise drift.

## What it provides

- Flat variant registry.
- Deep merge with array replacement.
- Inheritance.
- Runtime `app.config.ts` overrides.
- Generated TypeScript types.
- DevTools tab.
- Nuxt Content schema helper.
