`nuxt-variants` is a small Nuxt module for building one shared layout that can behave differently per page.

Use `nuxt-variants` when several pages share one layout but need typed layout options.

Keep those options in a registry instead of scattering route-specific layout logic across pages.

## What it provides

- Flat variant registry.
- Deep merge with array replacement.
- Inheritance.
- Runtime `app.config.ts` overrides.
- Generated TypeScript types.
- DevTools tab.
- Nuxt Content schema helper.
