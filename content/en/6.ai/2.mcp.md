# MCP

Docus exposes a built-in MCP server at `/mcp`.

Agents should use MCP to read current docs instead of guessing from stale memory.

```txt
MCP = retrieval of current documentation
Skills = behavior, workflow, and constraints
```

## Use MCP for

- Finding available docs pages.
- Reading the current version of architecture guidance.
- Checking product boundaries before editing.
- Confirming identity and Nuxt UI policy.

## Use skills for

- Activating the right workflow.
- Applying constraints.
- Remembering how to act while changing code.

MCP answers "what is true?" Skills answer "how should I act?"
